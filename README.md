# Overreaction

Forecasting Task Link:

https://pennmem.github.io/Overreaction/

## Files: 

- `index.html` — Forecasting task.
- `original_task.html` — Original forecasting task as designed in Afrouzi et al. (2023).
- `wordpool.js` — Contains the PEERS 1638 word pool as in Kahana et al. (2024).
- `data/` — Downloaded session JSON files here. Schema for the JSON files in **`data/data_schema.md`**.

To download data, open the task with **`?debug=1`** added to the URL, and then use the **Download data** button on the last screen.
